# .gitignore

## מה זה עושה
מגדיר אילו קבצים git יתעלם מהם – חוסם העלאת חוזי לקוחות, פלטים, וקבצי סביבה.

## תפקיד בפרויקט
חוסם בעיקר:
- `outputs/` – פלטי הניתוח (XLSX/DOCX עם נתוני לקוחות אמיתיים)
- `*.xlsx`, `*.docx` – למניעת דליפת מסמכים, **למעט** `data/**/*.docx` שמותרים
- `__pycache__/`, `.venv/`, `.env` – שגרת Python

## למי זה שייך
- **Owner**: Ops / Security
- **חיוני** – ללא הקובץ הזה, הסכמי לקוחות עלולים להגיע ל-GitHub בטעות
